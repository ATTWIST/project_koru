# J-Life Kit

Practical Japan-life tools for non-Japanese residents.

## Tools

- Monthly living-cost calculator
- Apartment move-in cost calculator
- Japan-life checklist
- Japanese document review prompt generator

## Public Site

The deployable static site is in:

```text
site/
```

GitHub Pages is configured through:

```text
.github/workflows/pages.yml
```

The app currently runs entirely in the browser. Uploaded images are previewed locally and are not sent to a server.
